# AngularTourOfHeroes

Este proyecto sigue el principio de responsabilidad única (SRP), uno de los principios SOLID más importantes. 
Basado en la abstracción de "componentes", trae consigo un conjunto de virtudes para nuestro código.

Beneficios:
- Todo es reutilizable,
- Cada componente se testea de forma abstraida
- Cada componente puede tener su propia "evolución"
- Mantener el proyecto en si mismo es mas sencillo dada la abstracción y separación en responsabilidades.
